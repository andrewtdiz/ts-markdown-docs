Install TS Markdown with the npm package `ts-markdown`

## Prerequisites

- **Node.js** 18+ or **Bun** 1.0+ (Bun recommended for best performance)
- **TypeScript** (optional, but recommended for type safety)

## Installation

TS Markdown works with any package manager. We recommend Bun for the fastest installation:

```shell
bun add tsmarkdown
```

Or with npm:
```shell
npm install tsmarkdown
```

That's it! TS Markdown has zero dependencies and works right out of the box.
